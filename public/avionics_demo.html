GLG Avionics Web Demo (HTML5 & JavaScript)

This is a pure HTML5 and JavaScript example that loads custom gauges created with the GLG Graphics Builder into a JavaScript program and animates them with real-time data using the GLG JavaScript Library API. More information

The drawing containing a panel of custom avionic gauges was created with the GLG Graphics Builder, which allows a user to create graphical objects and define their dynamic behavior using a point-and-click interface, without the need to write graphics code by hand. The drawing is saved into a file (avionics.g). Click here to see more avionic gauges and other available widgets.

The GLG JavaScript Library is used to deploy the drawing on a web page. The library provides an API to load and display the drawing, animate it with real-time data and handle user interaction.

The GLG JavaScript Library implements full GLG run-time functionality present in the GLG C/C++, C# and Java libraries, allowing the same graphical page and programming logic to be shared between the desktop, web and mobile versions of an application.
